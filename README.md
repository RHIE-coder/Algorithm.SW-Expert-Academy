### 빅-오(O) 표기법

 - 시간 복잡도 함수 중에서 가장 큰 영향력을 주는 n에 대한 항만을 표시
 - 계수(Coefficient)는 생략하여 표시

#### O(2n^2 + 10n + 100) = O(n^2)
